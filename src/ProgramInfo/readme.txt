HOW TO RUN THE PROGRAM:
- To run the main program, go to Main package and run the main method in ConferenceApp.
- Times are specified by military time, (01:00)
- Dates are specified in the form (DD/MM/YYYY)
- You must log out in order to save information, otherwise information will not be saved.
- All the .jar files in jarfiles folder in phase 2 need to be added to classpath for PDF schedule to work.
